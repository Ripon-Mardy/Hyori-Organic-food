# 🥬 Hyori Organic Food

Hyori Organic Food is a modern full-stack ecommerce platform for organic food products.
Built with Next.js, Node.js, Express.js, MongoDB, and Tailwind CSS.

---

## 🚀 Features

- Modern responsive UI
- Banner slider with Embla Carousel
- Product categories & subcategories
- Dynamic routing with Next.js App Router
- Authentication system
- Shopping cart & checkout
- Product management
- Admin dashboard
- REST API integration
- Mobile friendly design

---

## 🛠️ Tech Stack

### Frontend

- Next.js
- Tailwind CSS
- Embla Carousel
- Framer Motion
- Lucide React

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
