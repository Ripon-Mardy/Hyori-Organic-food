# 🥬 Hyori Organic Food

Hyori Organic Food is a modern full-stack ecommerce platform for organic food products.
Built with Next.js, Node.js, Express.js, MongoDB, and Tailwind CSS.

---

## 🚀 Features

- Modern responsive UI# ▶️ How To Run The Project With Docker

Follow these steps to run the project locally using Docker.

---

# 1️⃣ Clone The Repository

```bash
git clone https://github.com/YOUR_USERNAME/Hyori-Organic-food.git
```

---

# 2️⃣ Go To Project Folder

```bash
cd Hyori-Organic-food
```

---

# 3️⃣ Create Environment Files

## Frontend `.env.local`

Create file:

```bash
frontend/.env.local
```

Add:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_BASE_IMAGE_URL=http://localhost:5000
```

---

## Backend `.env`

Create file:

```bash
backend/.env
```

Add:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

# 4️⃣ Run Docker Containers

Make sure Docker Desktop or Docker Engine is running.

Run:

```bash
docker compose up --build
```

OR

```bash
docker-compose up --build
```

---

# 5️⃣ Open In Browser

Frontend:

```bash
http://localhost:3000
```

Backend API:

```bash
http://localhost:5000
```

---

# 🛑 Stop Containers

```bash
docker compose down
```

---

# 🔄 Rebuild Containers

```bash
docker compose up --build
```

---

# ✅ Requirements

Make sure these are installed:

- Docker
- Docker Compose
- Git

---

# 🐳 Docker Commands

## Show Running Containers

```bash
docker ps
```

---

## Remove Containers

```bash
docker compose down
```

---

## Remove Containers + Volumes

```bash
docker compose down -v
```

---

# 🔥 Production Build

```bash
docker compose up --build -d
```

Run in background mode.

---
